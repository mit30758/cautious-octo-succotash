# Fapte bune pe care le poți face

**În familie:** Poți ajuta părinții la treburile casnice ușoare sau poți pregăti o masă simplă.  
**La școală:** Poți împărți un creion sau o gumă de șters cu un coleg sau îl poți ajuta pe un prieten să înțeleagă o lecție.  
**În comunitate:** Poți zâmbi și saluta vecinii, poți participa la plantarea de copaci sau poți dona jucării și haine pentru copiii mai puțin norocoși.  
**Pentru natură:** Poți strânge gunoaiele dintr-un parc sau poți recicla materialele din casă.
